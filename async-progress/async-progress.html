An IXD prototype demonstrating a UI control that reflects target and reported state.

  1. What if we need a slider representing some value on a networked device configurable within a range?
  2. ...and we need to reflect the value the user is intending to change that value to (Target State)
  3. ...and we need to reflect the most recently reported value from the device (Reported State)

When increasing the value the difference between the target and reported state is in yellow.

When decreasing the value the difference between teh target and reported state is in red.
